<h2>Basic plots in R</h2>

R-base package graphics offers functions for producing many plots, for example:

* scatter plots - plot()
* dot plots - dotchart()
* histograms - hist()
* box plots - boxplot() 
* bar plots - barplot()
* pie charts - pie()
