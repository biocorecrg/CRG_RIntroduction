<h2>Data types</h2>

Each object has a data type:
* Numeric (number - integer or double)
* Character (text)
* Logical (TRUE / FALSE)

<h3>Checking data types</h3>

Number:
```{r}
a <- 10
mode(a)
typeof(a)
str(a)
```

Text:
```{r}
b <- "word"
mode(b)
typeof(b)
str(b)
```
